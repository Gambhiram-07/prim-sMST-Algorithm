# Minimum Spanning Tree (MST) Algorithm in Java

This repository contains the Java implementation of Minimum Spanning Tree (MST) algorithm using Prim's Algorithm.

## 📌 Description
Minimum Spanning Tree is used to connect all vertices in a graph with minimum total edge weight and without cycles.

## 🚀 Algorithm Included
- Prim's Algorithm

## 💻 Language Used
- Java
